HTML의 태그를 알아보자.

feat.생활코딩

  1. Normal Tag
  2. Attrbute Tag
  3. 부모-자식 Tag

1.Normal Tag

  • h1~h6 : 제목
  • strong : 글자 굵게
  • u : 밑줄
  • br : 줄 바꿈 (닫을 필요 없음)
  • p : 단락 바꿈
  • title : 웹의 제목
  • !doctype html : html의 문서입니다를 알려줌, 관용적 표현
  • html / head / body : html의 구조, 제목구조, 본문구조
  • a : 링크(닻의 연결)

2.Attribute Tag

  • img : 사진 삽입 (src : 사진위치 / width : 사진크기)
    • -> img src="~.jpg" width="~px/~%"
  • meta charst="~" : ~의 형식으로 열라고 주문, atom의 경우 utf-8 (파일 깨질때 사용)
  • a : 링크 (href="~" : 사용할 링크 / target='_blank' 새 창 / title="~" : 미리보기)
    • -> a href="~" target="_blank" title="~"

3.부모-자식 Tag

  • li - ul : 목록 - 그룹 (부모자식 태그)
  • ol : 자동 넘버링(그룹, ul파생)
