1. Normal Tag

  • h1~h6 : 제목
  • strong : 글자 굵게
  • u : 밑줄
  • br : 줄 바꿈 (닫을 필요 없음)
  • p : 단락 바꿈
  • title : 웹의 제목
  • !doctype html : html의 문서입니다를 알려줌, 관용적 표현
  • html / head / body : html의 구조, 제목구조, 본문구조
  • a : 링크(닻의 연결)
